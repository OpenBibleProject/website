## Features

<CardGrid stagger>
	<Card title="Easy to implement" icon="rocket">
		The OpenBibleProject aims to build a easy to implement specification for bible modules.
	</Card>
	<Card title="Portable" icon="puzzle">
		The bible modules are built to be portable, be easily downloaded and put anywhere you need it.
	</Card>
	<Card title="Easy to read" icon="translate">
		The modules consist of Markdown files that are easy to read without special software and can be used for various use-cases.
	</Card>
	<Card title="Open Source" icon="github">
		The Spec and all the Modules of the OpenBibleProject are [open source](https://github.com/OpenBibleProject) and free to use.
	</Card>
</CardGrid>
