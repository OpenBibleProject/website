## Features

<CardGrid stagger>
	<Card title="Einfache Implementierung" icon="rocket">
		Das OpenBibleProject zielt darauf ab, eine einfach zu implementierende Spezifikation für Bibelmodule zu erstellen.
	</Card>
	<Card title="Portabel" icon="puzzle">
		Die Bibelmodule sind so konzipiert, dass sie portabel sind, leicht heruntergeladen werden können und überall dort eingesetzt werden können, wo sie gebraucht werden.
	</Card>
	<Card title="Einfach zu lesen" icon="translate">
		Die Module bestehen aus Markdown-Dateien, die ohne spezielle Software leicht zu lesen sind und für verschiedene Anwendungsfälle genutzt werden können.
	</Card>
	<Card title="Open Source" icon="github">
		Die Spezifikation und alle Module des OpenBibleProjects sind [open source](https://github.com/OpenBibleProject) und frei zu verwenden.
	</Card>
</CardGrid>
